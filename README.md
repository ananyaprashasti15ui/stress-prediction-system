# StressPredict: Intelligent Stress Analysis System

## Overview
This project is a Python-based intelligent system designed to analyze and classify stress levels using a questionnaire-based approach inspired by research in stress detection and mental health analysis.

## Objectives
- Analyze stress levels using user responses  
- Provide explainable insights into stress factors  
- Visualize stress patterns using graphs  
- Simulate a basic mental health decision-support system  

## Features
- Psychologist-style interaction  
- Graph visualization using matplotlib  
- Weighted scoring system (novelty)  
- Explainable results (reason-based output)  
- Personalized suggestions  

## Methodology
1. User answers stress-related questions  
2. Responses are assigned weighted scores  
3. Total stress score is calculated  
4. Stress level is classified (Low / Moderate / High)  
5. Key factors affecting stress are identified  
6. Graph visualization is generated  

## Novelty
Unlike traditional systems, this project:
- Uses weighted scoring instead of simple classification  
- Provides explainable outputs  
- Includes graphical visualization  
- Combines rule-based and analytical approach  

## How to Run
